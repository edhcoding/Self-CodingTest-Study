알고리즘 문제를 풀기 위해서는 자료구조에 대한 이해가 필요합니다.

자료구조(Data Structure)란?
- 자료구조는 다수의 자료(data)를 담기 위한 구조 입니다.
- 데이터의 수가 많아질수록 효율적인 자료구조가 필요합니다.
ex) 학생 수가 1,000,000명 이상인 학생 관리 프로그램
학생 수가 너무 많기 때문에 학생수를 관리할 방법을 데이터의 관점에서 바라볼 필요가 있습니다.
컴퓨터에는 문자와 같은 형태로 기록할 수 있습니다.
수가 굉장히 많아지면 효율적인 자료구조로 이러한 데이터를 표현하지 않는다면 많은 메모리의 낭비가 존재할 수 있습니다.
특히, 학생 데이터를 검색하고 수정하고 삭제하고 추가하는 과정에서는 많은 문제가 발생할 수 있기 때문에 효율적인 자료구조를 선택해 적용하는게 매우 중요합니다.

자료구조 개요
- 성능비교: 자료구조 / 알고리즘의 성능 측정 방법에 대해 이해가 필요합니다.
어떤 자료구조를 선택하냐에 따라 수행시간이 달라질 수 있습니다.
ex)
1 번 케이스
삽입과 추출이 적당히 빠름
삽입: O(logN), 추출: O(logN)

2번 케이스
삽입은 느리지만 추출은 빨라
삽입: O(N), 추출: O(1)

N은 일방적으로 데이터의 개수를 의미합니다.
데이터 수가 적다면 어떤걸 사용해도 비슷하겠지만 데이터가 많아질수록 적절한 자료구조를 선택해 해당 이슈를 처리하는게 매우 중요합니다. 

자료구조와 알고리즘은 아주 밀접한 관계가 있습니다.
자료구조를 구현하기 위해서 알고리즘이 필요하고 알고리즘 문제를 해결하기 위해서 자료구조를 활요하는 경우가 많습니다.

자료구조의 필요성
- 자료구조를 제대로 이해하지 못하면 불필요하게 메모리와 계산을 낭비할 여지가 있습니다.

ex)
C언어를 기준으로 정수(int) 형식의 데이터가 100만 개가량이 존재한다고 가정해보자
해당 프로그램을 이요하면, 내부적으로 하루에 데이터 조회가 1억 번 이상 발생합니다.
이때 원하는 데이터를 가장 빠르게 찾도록 해주는 자료구조는 무엇일까? - 트리(tree)와 같은 자료구조를 활용할 수 있습니다.

자료구조의 종류
1. 선형 구조(linear data structure)
- 배열 (Array)
- 연결 리스트 (linked list)
- 스택 (stack)
- 큐 (queue)

2.비선형 구조(non-linear data structure)
- 트리 (tree)
- 그래프 (graph)

선형 자료구조 (Linear Data Structure)
- 선형 자료구조는 하나의 데이터 뒤에 다른 데이터가 하나 존재하는 자료구조 입니다.
- 데이터가 일렬로 연속적으로(순차적으로) 연결되어 있습니다.
ex) 배열, 연결 리스트(linked list), 스택(stack), 큐(queue)

비선형 자료구조(Non-Linear Data Structure)
- 비선형 자료구조는 하나의 데이터 뒤에 다른 데이터가 여러 개 올 수 있는 자료구조 입니다.
- 데이터가 일직선상으로 연결되어 있지 않아도 됩니다.
ex) 트리(tree), 그래프(graph)

프로그램의 성능 측정 방법
- 시간 복잡도(time complexity): 알고리즘에 사용되는 연산 횟수를 측정합니다.
- 공간 복잡도(space complexity): 알고리즘에 사용되는 메모리의 양을 측정합니다.
- 공간을 많이 사용하는 대신 시간을 당축하는 방법이 흔히 사용됩니다.

프로그램의 성능 측정 방법: Big-O 표기법
- 복잡도를 표현할 때는 Big-O 표기법을 사용합니다.
1) 특정한 알고리즘이 얼마나 효율적인지 수치적으로 표현할 수 있습니다.
2) 가장 빠르게 증가하는 항만을 고려하는 표기법입니다.
- 가장 큰 항만을 표기합니다. (가장 큰 항에 붙어있는 계수는 제거합니다)

일방적인 코딩 테스트 채정용 서버를 기준으로 연산 횟수가 10억을 넘어가면 1초 이상이 소용된다고 합니다.
ex) n이 1000일 때 고려해보자
O(n) - 약 1000번의 연산
O(nlogn) - 약 10,000번의 연산
O(n^2) - 약 1,000,000번의 연산
O(n^3)  - 약 1,000,000,000번의 연산
코딩 테스트에서 메모리의 크기를 나타낼 때는 일반적으로 MB단위로 표기합니다.
